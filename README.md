# Campus-Network-Design
A Cisco Packet Tracer project for designing and implementing a secure university campus network with VLANs, DHCP, dynamic routing, wireless connectivity, and network security.
## Features
- Hierarchical Network Design
- VLAN Segmentation
- Dynamic Routing
- DHCP Configuration
- Wireless Connectivity
- Network Security
- Inter-VLAN Routing

## Tools Used
- Cisco Packet Tracer
- Cisco Routers
- Cisco Switches

## Project Goal
The goal of this project is to design a scalable, secure, and efficient campus network for a university using networking best practices.
